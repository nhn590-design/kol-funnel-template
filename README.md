# KOL AI Funnel Template

Template funnel tuyển thành viên + onboarding gamified (landing page → dashboard tiến trình theo tuần), tự chứa trong một file `index.html`, không cần build.

## Live Site
https://nhn590-design.github.io/kol-funnel-template/

## Trước khi dùng thật
Thay toàn bộ nội dung trong `{{ }}` ở `index.html` (tên chương trình, link Zalo/Telegram, link video từng phần) bằng nội dung thật.

## Deploy
Tự động qua GitHub Actions — push lên `main` là deploy lại.
